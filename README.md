# Eva-CAM

## Overview
Eva-CAM is a circuit/architecture-level modeling and evaluation tool for content addressable memories (CAMs). Eva-CAM supports Ternary CAM, Aanalog CAM, and Multi-bit CAM designs implemented in non-volatile memories, for both exact and approximate match types. It also allows for the exploration of CAM array structures and sensing circuits. Eva-CAM has been validated with HSPICE simulation results and chip measurements.
